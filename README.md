* sm_beaconlasthuman_time -Default 30- The amount of time in seconds to beacon last survivor.
* sm_beaconlasthuman_color -Default 0- Color beacon [ 0=Red 2=Orange 3=Green 4=Blue 5=White 6=Grey ]
